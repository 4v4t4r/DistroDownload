# DistroDownload
面向中国的发行版下载仓库

## 介绍
本仓库旨在通过网盘分享的形式为广大开发者提供开源镜像下载，以解决部分发行版下载困难问题。所有镜像均来自官方，并提供对应SHA1以及MD5码，请自行进行校验。

## 使用
使用`Ctrl+F`搜索想要下砸的发行版，进入对应文档，下载对应版本即可。

下载完成后，建议使用Hashcalc进行SHA1以及MD5校验。

### 镜像发布
提供大容量网盘分享，要求无广告，无密码，无短连接中转，长期稳定。

### 镜像校验
负责校验已有的镜像完整性及安全性。
